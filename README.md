# AI Story Generator 📖

## Overview

This is a simple AI-based story generator project.
It uses a Large Language Model (LLM) to create stories based on user input.

---

## Features

* Generate stories from prompts
* Supports different genres (fantasy, horror, etc.)
* Easy to use
* Can adjust story length

---

## Tech Stack

* Python
* Google API (OpenAI / Hugging Face)

---

## Future Improvements

* Add UI
